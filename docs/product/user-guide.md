# User Guide

## Reading the Shell

Use the shell to confirm:

- organization
- entity
- module
- period

before taking action.

## Intent Panel

Use the intent panel to inspect:

- intent identifier
- status
- validation results
- approval chain
- job identifier
- record references

## Job Panel

Use the job panel to review:

- running jobs
- completed jobs
- failed jobs
- backend error messages

## Airlock

Use the airlock queue and review screens to:

- see uploaded items
- inspect validation issues
- admit clean items
- reject invalid items with a reason

## Journals

Use the journal list to verify:

- intent linkage
- job linkage
- approval status

before opening or progressing a journal.

## Traceability

Use the control-plane evidence surfaces for:

- timeline review
- snapshot navigation
- lineage inspection
- impact preview
- audit export
